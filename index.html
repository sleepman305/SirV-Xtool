Ngôn ngữ
PDF có text sẵn (sinh từ Word, web...) → trích thẳng, nhanh và chính xác 100%.

Độ phân giải
Cách hoạt động: mỗi trang PDF được render lại thành ảnh JPEG rồi gộp lại thành PDF mới. Hiệu quả nhất với PDF scan và PDF có nhiều ảnh.

Chọn nguồn
⚠ YouTube: Cobalt server bị YouTube chặn. Cần cài bộ riêng:
  1. Cobalt (TikTok/IG/FB/X): Cài shortcut
  2. YouTube cần 3 thứ:
    • App Scriptable
    • App a-Shell mini
    • YouTube Download shortcut
Cách 1 — Mở dịch vụ tải
Cách 2 — Dùng yt-dlp (terminal)
Cách 1 hoạt động thế nào:
  1. Dán link video vào ô trên
  2. Bấm 1 nút bất kỳ ở trên
  3. URL tự động copy vào clipboard, trang dịch vụ mở ra ở tab mới
  4. Paste URL vào ô của trang đó → tải về
Cách 2: Cần cài yt-dlp. Trên Windows: pip install yt-dlp rồi paste lệnh đã copy vào CMD/PowerShell.

Kiểu tóm tắt
Chưa có API key. Vào tab 🤖 AI → API key để thiết lập (miễn phí từ Google AI Studio).
Cấu hình AI
Gemini API Key
Model
Lấy API key miễn phí:
  1. Vào aistudio.google.com/apikey
  2. Đăng nhập Google → bấm "Create API key"
  3. Copy key (bắt đầu bằng AIza...)
  4. Dán vào ô trên → Lưu
Quota miễn phí: 15 yêu cầu/phút, 1500/ngày. Đủ dùng cá nhân.
Bảo mật: Key lưu cục bộ trên máy (localStorage), không gửi đi đâu trừ trực tiếp đến Google.
